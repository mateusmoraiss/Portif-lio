Projetos

Programa criado em C para pizzaria

Tive a oportunidade de dar meus primeiros passos na programação ao criar um programa em C para uma pizzaria. Eu tinha conhecimentos básicos em alguns comandos como printf, if else, switch case, do while , entre outros. Apesar disso, desenvolvi todo o programa (de 1250 linhas) sozinho, com exceção da ajuda que recebi do meu grupo de cinco pessoas, que me auxiliaram com os nomes e preços das pizzas e outras informações relevantes. Foi uma experiência desafiadora, mas muito gratificante.

Para ser mais preciso, eu fui responsável por programar toda a parte lógica do programa em C para a pizzaria, incluindo a criação dos algoritmos e estruturas de dados necessárias para que o programa pudesse funcionar corretamente. Meus colegas de grupo me ajudaram apenas com alguns detalhes. Mas, em termos de programação, eu fui o principal responsável pelo desenvolvimento do programa.
